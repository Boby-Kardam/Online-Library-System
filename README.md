Online Library - React Application
Overview
A simple online library application built with React, Redux, and React Router. This application allows users to:

Browse books by category

View book details

Search for books

Add new books to the library

Features
Home Page: Displays book categories and popular books

Category View: Shows books filtered by category with search functionality

Book Details: Displays comprehensive information about a selected book

Add Book: Form to add new books to the library

404 Handling: Custom page for invalid routes
